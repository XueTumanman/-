# 1. 线性表的定义

**说明**

1. 相同数据类型的有限序列
2. 逻辑上有顺序性，有先后次序

>**只是一种逻辑结构**

**专业术语**

| 名称 | 说明 |
|---|---|---|
| 表头元素 | 第一个数据元素 |
| 表尾元素 | 最后一个数据元素 |
| 直接前驱 | 前一个元素 |
| 直接后继 | 后一个元素 |
