# 1. 用户接口

- ## 联机用户接口

  **说明**
  1. 组成 = 一组键盘命令 + 命令解释程序

  **过程**
  1. 用户在【终端，控制台】输入命令
  2. 系统转入命令解释程序
  3. 执行完指定功能结果返回【终端，控制台】
  4. 等待下一条命令

- ## 脱机用户接口

  **说明**
  1. 为批处理作业提供

  **过程**
  1. 用作业控制语言 JCL 把要对作业进行的控制命令事先写在作业说明书上
  2. 提供给 OS
  3. OS 调用到作业时，使用命令解释程序对作业说明书上的命令逐条解释执行
  4. 遇到作业结束语句 OS 停止作业的运行

- ## 图形用户接口

  **说明**
  1.简化操作，不用键入命令

# 2. 程序接口

**说明**
1. 为用户程序执行过程中访问 OS 资源而设置
2. 用户程序取得 OS 服务的唯一途径
3. 组成 = 一组系统调用（能完成特定功能的子程序）
